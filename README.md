# C Color Library
**ccl**, for **C Color Library** is a simple, header-only, single-file, some other things â€” C library for colored prompts

## API
- `ccl.fg("#0f0f0f")`: *true*color HEX code for the foreground
- `ccl.bg("#0f0f0f")`: *true*color HEX code for the background
- `ccl.fg_ansi("red")`: `white` or any other standard ANSI color
- `ccl.printf("#287233, "", ...)`: First argument is a *true*color HEX code, then standard `fmt` and `va_args`
- `ccl.reset()`: Resets **all** colors
- `ccl.style("bold")`: Applies a **st****yle*** to the text.
