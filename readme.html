API

  • ccl.fg("#0f0f0f"): truecolor HEX code for the foreground.
  • ccl.bg("#0f0f0f"): truecolor HEX code for the background.
  • ccl.fg_ansi(red): white or any other standard ANSI color
  • ccl.printf("#287233, "", ...): First argument is a truecolor HEX code, then standard fmt and va_args
  • ccl.bg("#0f0f0f")
